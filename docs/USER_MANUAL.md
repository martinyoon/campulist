## 1. 서비스 개요

캠퍼스리스트는 대학교 재학생, 교직원, 캠퍼스 주변 상인을 위한 로컬 커뮤니티 플랫폼입니다.

| 연세대학교 | 연세대 | 서울 신촌 | yonsei.ac.kr |
| 고려대학교 | 고려대 | 서울 안암 | korea.ac.kr |

| 한예종 | 한예종 | 서울 석관 | karts.ac.kr |

### 회원 유형 (7종)

| 유형 | 아이콘 | 설명 |
|------|--------|------|
| 학부생 | 🎓 | 대학교 재학 중인 학부생 |

| 일반인 | 👤 | 위 분류에 해당하지 않는 일반 사용자 |

## 2. 회원가입 및 로그인

**경로**: `/auth`

### 로그인

1. 상단 헤더의 **로그인** 버튼을 클릭합니다.
2. 이메일과 비밀번호를 입력합니다.
3. **로그인** 버튼을 클릭합니다.
4. 성공 시 "로그인 되었습니다" 메시지와 함께 홈으로 이동합니다.

### 회원가입

1. 로그인 화면에서 **회원가입** 탭을 선택합니다.
2. 다음 정보를 입력합니다:
   - **닉네임**: 사용할 닉네임
   - **회원 유형**: 7종 중 선택 (학부생, 대학원생, 교수, 교직원, 졸업생, 인근상인, 일반인)
   - **이메일**: 학교 이메일 권장
   - **비밀번호**: 비밀번호 설정
3. **이메일 자동 인증**: `.ac.kr` 도메인 이메일(예: `abc@snu.ac.kr`)로 가입하면 해당 대학교가 자동으로 인증됩니다.
4. **일반 이메일 가입**: `.ac.kr` 이외 이메일로 가입 시 "관련 대학교"를 직접 선택해야 합니다.
5. **회원가입** 버튼을 클릭하면 즉시 로그인되어 홈으로 이동합니다.

### 소셜 로그인 (준비 중)

- Facebook, 카카오, 네이버 소셜 로그인은 현재 준비 중입니다.

## 3. 홈 화면

**경로**: `/`

홈 화면은 다음 섹션으로 구성됩니다:

### 대학 선택 탭
- 화면 상단에 5개 대학교 탭이 표시됩니다.
- 탭을 클릭하면 해당 대학교 전용 페이지로 이동합니다.
- 홈에서는 "모든 대학" 통합 피드를 보여줍니다.

### 카테고리 바로가기
- 7개 대분류 카테고리가 아이콘 그리드로 표시됩니다.
- 클릭하면 해당 카테고리의 전체 대학 게시글을 볼 수 있습니다.

### 실시간 인기글
- 조회수/좋아요 기준 인기 게시글 5건이 표시됩니다.
- 접기/펴기 토글이 가능합니다.

### 캠퍼스라이프
- 캠퍼스 주변 맛집, 할인, 이벤트 등의 게시글이 가로 스크롤 카드로 표시됩니다.
- **더보기** 링크로 전체 목록을 볼 수 있습니다.

### 최신 게시글
- 전체 대학의 최신 게시글이 목록으로 표시됩니다.
- 로컬에서 작성한 게시글도 함께 표시됩니다.

## 4. 대학교별 페이지

**경로**: `/{대학교슬러그}` (예: `/snu`, `/yonsei`, `/korea`, `/kaist`, `/karts`)

### 대학교 전용 피드
- 선택한 대학교의 게시글만 필터링하여 표시합니다.
- 대학 정보 배너에 대학교명, 지역, 영문명이 표시됩니다.

### 정렬 옵션
- **최신순** (기본값)
- **가격 낮은순**
- **가격 높은순**
- **인기순**

### 대학교별 카테고리
- 카테고리 그리드에서 클릭하면 `/{대학교}/{카테고리}` 페이지로 이동합니다.
- 해당 대학교 + 해당 카테고리의 게시글만 표시됩니다.

## 6. 게시글 작성

**경로**: `/write` (로그인 필요)

### 작성 흐름 (2단계)

**1단계: 카테고리 선택**
- 대학교 선택 (기본값: 본인 소속 대학)
- 대분류 > 소분류를 한 화면에서 선택
- 소분류를 클릭하면 바로 글쓰기 폼으로 이동

**2단계: 글쓰기 폼**

| 필드 | 필수 | 설명 |
|------|------|------|
| 대학교 | O | 어느 대학 게시판에 올릴지 선택 |
| 카테고리 | O | 대분류 + 소분류 (변경 가능) |
| 제목 | O | 최대 100자, 자동으로 `[대학약칭][회원유형] ` 접두어 삽입 |
| 가격 | X | 숫자 입력 (원), "가격 협의 가능" 체크 가능 |
| 내용 | O | 최대 5,000자 |
| 사진 | X | 최대 10장, 자동 압축(800px, JPEG 70%), 첫 번째 사진이 "대표" 썸네일 |
| 태그 | X | 최대 5개, 입력 후 추가 버튼 또는 Enter |
| 거래 희망 장소 | X | 텍스트 입력 (예: "서울대 정문 GS25 앞") |
| 연락 방법 | - | 캠톡(기본), 전화번호, 카카오 오픈채팅, 이메일 중 선택 |

### 예시 채우기 기능
- 소분류별로 미리 준비된 예시가 있습니다.
- **"전체 예시 채우기"** 버튼으로 모든 필드를 한 번에 채울 수 있습니다.
- 개별 필드의 **"예시"** 링크로 필드별 예시도 적용 가능합니다.

### 임시저장
- 글쓰기 중 내용이 **1초 간격으로 자동 임시저장**됩니다.
- 다음 방문 시 임시저장된 내용이 자동 복원됩니다.
- **초기화** 버튼으로 임시저장 삭제 가능합니다.

### 카테고리 페이지에서 글쓰기
- 대학교 페이지나 카테고리 페이지에서 "글쓰기" 버튼을 누르면 해당 대학교와 카테고리가 자동 선택됩니다.

### 게시글 수정
- 마이페이지의 "내 게시글" 탭에서 **수정** 버튼을 클릭합니다.
- 또는 게시글 상세 페이지에서 수정 기능을 이용합니다.
- 수정 모드에서는 **거래 상태**(판매중/예약중/거래완료)를 변경할 수 있습니다.
- **삭제하기** 버튼으로 게시글을 영구 삭제할 수 있습니다 (복구 불가).

## 7. 게시글 상세

**경로**: `/post/{게시글ID}`

### 화면 구성

**이미지 갤러리**
- 등록된 사진을 스와이프하여 볼 수 있습니다.

**작성자 정보**
- 프로필 이미지(닉네임 이니셜), 닉네임
- 학교 인증 뱃지 (인증된 경우)
- 매너온도, 거래 횟수
- 클릭하면 작성자 프로필 페이지로 이동

**브레드크럼 내비게이션**
- `대학교 > 대분류 > 소분류` 형태로 표시
- 각 단계를 클릭하면 해당 페이지로 이동
- 노란색 강조 배경으로 구분

**게시글 내용**
- 거래 상태 뱃지 (예약중/거래완료)
- 제목, 본문
- 작성일, 조회수, 찜 수
- 태그 (클릭 시 해당 태그로 검색)
- 거래 희망 장소 (설정된 경우)
- 연락 방법 (전화번호, 카카오, 이메일 등)

**하단 고정 바**
- **찜 버튼** (하트): 게시글을 찜 목록에 추가/제거
- **공유 버튼**: 게시글 링크 복사
- **가격 표시**: 가격 및 협의 가능 여부
- **캠톡하기 버튼**: 작성자에게 캠톡 메시지 전송
- **끌어올리기** (본인 게시글): 게시글을 최신으로 올림

**관련 게시글**
- 같은 카테고리의 다른 게시글 최대 4개 표시

### 거래 상태 변경 (작성자만)
- 본인 게시글에서 "판매중 → 예약중 → 거래완료" 상태를 변경할 수 있습니다.

### 신고 기능
- 게시글 우측 상단의 신고 버튼으로 부적절한 게시글을 신고할 수 있습니다.

## 8. 검색

**경로**: `/search`

### 검색 방법
1. 상단 헤더의 검색창에 검색어를 입력합니다.
2. 하단 네비게이션의 **검색** 탭을 클릭합니다.
3. 검색어를 입력하면 제목, 본문, 태그에서 매칭되는 게시글을 표시합니다.

### 정렬 옵션
- **최신순** (기본)
- **가격 낮은순**
- **가격 높은순**
- **인기순**

### 가격 필터
- 최소 가격 ~ 최대 가격 범위를 설정하여 결과를 필터링합니다.

### 최근 검색어
- 검색어가 없을 때 최근 검색어 목록이 표시됩니다.
- 개별 검색어 삭제 또는 전체 삭제가 가능합니다.

## 9. 캠톡 (실시간 채팅)

**경로**: `/camtalk` (로그인 필요)

캠톡은 캠퍼스리스트의 1:1 메시징 시스템입니다.

### 캠톡 시작하기

**게시글에서 시작**
1. 게시글 상세 페이지에서 **캠톡하기** 버튼을 클릭합니다.
2. 빠른 메시지를 선택하거나 직접 입력합니다:
   - "구매 의사가 있습니다. 거래 가능한가요?"
   - "가격 네고 가능한가요?"
   - "직거래 장소와 시간이 어떻게 되나요?"
   - 직접 입력하기
3. 메시지를 전송하면 캠톡 대화방이 생성되고 자동으로 이동합니다.

**프로필에서 시작**
- 사용자 프로필 페이지에서 **캠톡하기** 버튼을 클릭하면 해당 유저와의 대화방으로 이동합니다.

### 캠톡 목록
- 참여 중인 모든 캠톡 대화방이 표시됩니다.
- 각 방에 상대방 닉네임, 마지막 메시지, 시간이 표시됩니다.
- 안읽은 메시지가 있으면 숫자 뱃지가 표시됩니다.

### 캠톡 대화방
- 상단에 상대방 닉네임, 인증 뱃지, 매너온도가 표시됩니다.
- 닉네임을 클릭하면 상대방 프로필로 이동합니다.
- 메시지는 보낸 사람(파란색, 우측) / 받은 사람(회색, 좌측)으로 구분됩니다.
- 날짜가 바뀌면 날짜 구분선이 표시됩니다.
- 메시지 내 게시글 링크(`/post/xxx`)가 자동으로 클릭 가능한 "게시글 보기" 링크로 변환됩니다.
- 대화방 입장 시 자동으로 읽음 처리됩니다.

### 안읽음 알림
- 하단 네비게이션의 **캠톡** 아이콘에 빨간색 안읽음 숫자가 표시됩니다.
- 안읽음 수는 유저별로 정확하게 추적됩니다 (보낸 사람에게는 안읽음 표시 없음).

### 중요 규칙
- 같은 두 사용자 간에는 **하나의 캠톡 방만** 존재합니다.
- 이미 대화방이 있으면 새 방을 만들지 않고 기존 방으로 이동합니다.

|------|------------|------|
| camtalk | 파란색 | 캠톡 새 메시지 알림 |
| system | 회색 | 시스템 알림 |

### 알림 기능
- **읽지 않은 알림**: 파란색 배경 강조 + 파란색 점 표시
- **안읽음 개수**: 헤더 우측 "캠알림" 버튼에 빨간색 뱃지로 표시
- **개별 읽음**: 알림을 클릭하면 자동으로 읽음 처리되고 해당 링크로 이동
- **전체 읽음**: "전체 읽음" 버튼으로 모든 알림을 한 번에 읽음 처리

### 알림 원칙
- 알림은 **수신자(`recipientId`)별**로 관리됩니다.
- 본인에게 온 알림만 표시됩니다.
- 메시지를 보낸 사람에게는 알림이 발생하지 않습니다.

- **매너온도**: 거래 신뢰도 지표 (기본 36.5°, 높을수록 좋음)

### 메뉴

- **서비스 소개**: About 페이지로 이동
- **회원탈퇴**: 계정 삭제 (확인 절차 필요)

### 탭 (4개)

| 탭 | 설명 |
|----|------|
| **내 게시글** | 내가 작성한 게시글 목록 (상태 뱃지, 수정 버튼 포함) |
| **찜한 목록** | 하트 버튼으로 찜한 게시글 목록 |
| **최근 본** | 최근에 열어본 게시글 목록 |
| **받은 후기** | 거래 상대방이 남긴 후기 (별점 + 내용) |

### 회원탈퇴

2. 하단 시트에서 삭제되는 데이터 목록을 확인합니다:
   - 작성한 게시글 및 캠톡 내역
   - 찜한 목록 및 검색 기록
   - 알림 및 임시저장 데이터
3. 확인란에 **"탈퇴합니다"**를 정확히 입력합니다.
4. **회원탈퇴** 버튼을 클릭합니다.
